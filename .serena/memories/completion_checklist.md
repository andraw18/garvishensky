# Completion checklist

When a task is finished on this site:
- Verify the affected page(s) in a browser or Live Server preview.
- Check that desktop and mobile layouts still behave correctly.
- Confirm links, buttons, and any JS interactions still work.
- If only content changed, make sure the edited copy still matches the visual layout.
- If styles changed, inspect both `index.html` and `pages/services.html` because they share visual language but use separate stylesheets.
