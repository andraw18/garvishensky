# GARVISHENSKY project overview

GARVISHENSKY is a static Russian-language promotional website for a production center. The repo contains a main landing page (`index.html`) and a separate services page (`pages/services.html`). The site uses plain HTML, CSS, and JavaScript with local assets for images, video, icons, and interactive UI behavior.

## Main structure
- `index.html` - homepage
- `pages/services.html` - services/pricing page
- `assets/css/main.css` - homepage styles
- `assets/css/services.css` - services page styles
- `assets/js/main.js` - homepage interactivity
- `assets/js/services.js` - services page interactivity
- `assets/media/` - images and video assets
- `assets/icons/logo.svg` - logo asset

## Runtime / stack
- Static HTML/CSS/JS
- No build system detected
- No package manager or framework config detected
- External Google Fonts are loaded from the HTML files
