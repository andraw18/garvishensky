# Code style and conventions

- Use plain semantic HTML with section comments to separate page blocks.
- CSS is organized by page-specific stylesheet and written in a compact, utility-like style with custom properties in `:root`.
- JavaScript is handwritten vanilla JS, likely DOM-driven and event-listener based.
- Filenames use lowercase with hyphens or descriptive names as needed.
- Copy on the site is in Russian.
- Visual style is bold, editorial, and high-contrast, with prominent typography and accent red colors.
- When editing, preserve the existing class naming and layout structure unless the user asks for a redesign.
